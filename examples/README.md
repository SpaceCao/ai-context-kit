# Examples

This directory contains minimal examples that show how to apply `AI Context Kit` in a real project structure.

## Included example

- `minimal-design-repo/` — a tiny design-phase repository with:
  - `.ai-context/project-summary.md`
  - `.ai-context/topic-card-checkout.md`
  - `.ai-context/task-pack-checkout-flow.md`
  - `docs/checkout-prd.md`

Use it as a starter reference, not as a strict schema.
