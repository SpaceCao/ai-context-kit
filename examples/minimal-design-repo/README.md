# Minimal Design Repo Example

This example shows the smallest useful setup for running design work with `AI Context Kit`.

## Structure

```text
minimal-design-repo/
├── .ai-context/
│   ├── project-summary.md
│   ├── topic-card-checkout.md
│   └── task-pack-checkout-flow.md
└── docs/
    └── checkout-prd.md
```

## How to use this example

1. Read `.ai-context/project-summary.md`
2. Read `.ai-context/task-pack-checkout-flow.md`
3. Read `.ai-context/topic-card-checkout.md`
4. Only if needed, open `docs/checkout-prd.md`

That order mirrors the intended low-token workflow.
