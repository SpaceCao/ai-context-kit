# Prompt Template

把下面这段作为 Claude / Codex / ChatGPT 的任务前置提示：

```text
你现在只基于以下上下文工作：
1. project-summary
2. 当前 task-pack
3. 指定的 topic-card
4. 检索返回的少量原文 chunk

执行规则：
- 不要默认读取整个仓库
- 不要要求我粘贴整份长文档，除非当前证据不足
- 结论必须区分：已确认 / 高概率 / 待确认
- 如果现有上下文不足，先输出“补充检索建议”，不要直接扩写
- 如果存在目录路由或任务边界冲突，以 task-pack 为准

你的目标是：在最少上下文下完成当前设计任务。
```

推荐每次真正发问时，再补一句：

```text
本次只解决当前任务，不做跨模块发散。
```
